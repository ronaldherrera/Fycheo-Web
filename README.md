# Fycheo Web

Landing page pública y sitio web de presentación del producto Fycheo.

## Tecnologías
- **Frontend:** React, TypeScript, Vite.
- **Estilos:** CSS Modules / TailwindCSS (según configuración).

## Configuración Local

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar servidor:
   ```bash
   npm run dev
   ```
   
   La web se iniciará en el puerto **3002** y se expondrá a la red local automáticamente:
   - **Local:** `http://localhost:3002`
   - **Red:** `http://[IP-LOCAL]:3002`
